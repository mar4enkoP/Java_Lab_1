Task 2(61)

   Составить программу возведения натурального числа в квадрат, учитывая следующую закономерность:
   1^2 = 1
   2^2 = 1 + 3
   3^2 = 1 + 3 + 5
   4^2 = 1 + 3 + 5 + 7
   …
   n^2 = 1 + 3 + 5 + 7 + 9 + ... + 2n – 1
   
Task 3(86)

   б) все элементы соединены параллельно.
